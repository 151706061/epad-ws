web-based platform for quantitative imaging in the clinical workflow

REST API

RESTful Web Services Application Programmer Interface (API)

Sessions

/session [ POST: create a session using HTTP Basic authentication ]
/session/{id} [ DELETE: delete a session ]

Projects

/projects[?annotationCount=false] [ GET: get all projects. if annotationCount=false is passed, the annotation count in project objects will be zero(saves time) ]
/projects/{id}?projectName=[name]&projectDescription=[description] [ PUT: update an existing project ]
/projects/{id}?projectName=[name]&projectDescription=[description] [ POST: create a new project ]
/projects/{id} [ DELETE: remove a project ]
/projects/{id}/aims/?start=[index]&count=[number]&format=[summary | json] [ GET: get the annotations for the project. If no format is specified, the xml annotation will be returned. ]
/projects/{id}/aims/{id} [ PUT: create a new annotation ]

Subjects

/projects/{id}/subjects/ [ GET: get the project's subjects ]
/projects/{id}/subjects/{id}?subjectName=[name] [ PUT: create a subject in the project ]
/projects/{id}/subjects/{id}/status/{status} [ PUT: set status of the subject for this user and project ]
/projects/{id}/subjects/{id} [ GET: get a subject in the project ]
/projects/{id}/subjects/{id} [ DELETE: remove a subject in the project ]
/projects/{id}/subjects/{id}/aims/?start=[index]&count=[number]&format=[summary | json] [ GET: get the annotations for the subject in the project. If no format is specified, the xml annotation will be returned. ]
/projects/{id}/subjects/{id}/aims/{id} [ PUT: create an annotation ]
/projects/{id}/subjects/{id}/aims/{id} [ GET: get the annotation ]
/projects/{id}/subjects/{id}/aims/{id}?deleteDSO=true [ DELETE: remove the annotation ]

Studies

/projects/{id}/subjects/{id}/studies/ [ GET: get the study collection ]
/projects/{id}/subjects/{id}/studies/{id} [ PUT: add a study to the collection]
/projects/{id}/subjects/{id}/studies/{id} [ GET: get a study from the collection]
/projects/{id}/subjects/{id}/studies/{id}?deleteAims=true [ DELETE : delete a study from the collection]
/projects/{id}/subjects/{id}/studies/{id}/aims/?start=[index]&count=[number]&format=[summary | json] [ GET: get the annotation collection. If no format is specified, the xml annotation will be returned. ]
/projects/{id}/subjects/{id}/studies/{id}/aims/{id} [ PUT: create a new annotation in the collection ]
/projects/{id}/subjects/{id}/studies/{id}/aims/{id} [ GET: get the annotation in the collection ]
/projects/{id}/subjects/{id}/studies/{id}/aims/{id}?deleteDSO=true [ DELETE: remove the annotation in the collection ]

Series

/projects/{id}/subjects/{id}/studies/{id}/series/?filterDSO=true [ GET: get the series collection ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id} [ GET: get the series ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}?deleteAims=true [ DELETE: delete the series ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/aims/?start=[index]&count=[number]&format=[summary | json] [ GET : get the annotation collection. If no format is specified, the xml annotation will be returned. ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/aims/{id} [ PUT: create an annotation on the series]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/aims/{id} [ GET: get the annotation on the series. ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/aims/{id}?deleteDSO=true [ DELETE: remove the annotation from the series]

Images

/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/ [ GET: get the images in the series ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id} [ GET: get an image in the collection ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id} [ DELETE: delete an images in the collection ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/aims/?start=[index]&count=[number] [ GET: get the annotations IDs for the image ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/aims/{id} [ PUT: create an annotation for the image ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/aims/{id} [ GET: get the annotation ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/aims/{id}?deleteDSO=true [ DELETE: remove the annotation ]

Frames

/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/frames [ GET: get the frame collection]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/frames?type=new [ POST: modify the frame collection with new data]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/frames/{frame_number} [ GET: get the frame in the collection ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/frames/{frame_number}/aims/?start=[index]&count=[number]&format=[summary | json] [ GET: get the annotations for the frame. If no format is specified, the xml annotation will be returned. ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/frames/{frame_number}/aims/{id} [ PUT: create an annotation for the frame ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/frames/{id}/aims/{id} [ GET: get the annotation for the frame ]
/projects/{id}/subjects/{id}/studies/{id}/series/{id}/images/{id}/frames/{id}/aims/{id}?deleteDSO=true [ DELETE: remove the annotation for the frame ]

/subjects/ [ GET: get the subjects collection ]
/subjects/{id} [ GET: get the subject ]
/subjects/{id}/aims/?start=[index]&count=[number]&format=[summary] [ GET: get the annotations for the subject ]

Studies Shortcut

/studies/{id}/series [ GET: get the series collection ]
/studies/{id}/series/{id}/images/{id} [ GET: get the image ]
/studies/{id}/series/{id}/images/{id}/aims?start=[index]&count=[number]&format=[summary] [ GET: get the annotations for the image ]
/studies/{id}/series/{id}/images/{id}/frames [ GET: get the frame collection ]

Annotations Shortcut

/aims/?start=[index]&count=[number]&format=[summary] [ GET: get the annotations in the collection ]
/aims/{id} [ PUT: create the annotation ]
/aims/{id} [ GET: get the annotation ]
/aims/{id}?deleteDSO=true [ DELETE: remove the annotation ]

Files

/projects/{id}/subject/{id}/files/ [ GET: get non-DICOM files for this project and subject ]
/projects/{id}/subject/{id}/studies/{id}/files/ [ GET: get non-DICOM files for this project, subject and study]

/projects/{id}/subjects/{id}/studies/{id}/series/{id}/files/{filename} [ DELETE: delete a non-DICOM file for this series]

Remote PACs

Not yet implemented

/pacs/ [ GET: get configured Remote PACs in the system ]
/pacs/{id} [ GET: get configuration for this Remote PAC ]
/pacs/{id} [ DELETE: delete configuration for this Remote PAC ]
/pacs/{id}?aeTitle=[aeTitle]&hostname=[hostname]&port=[port]&primaryDeviceType=[deviceType] [ PUT: create/edit configuration for this Remote PAC ]
/pacs/{id}/entities/?patientNameFilter=[filter]&patientIDFilter=[filter]&studyDateFilter=[YYYYMMDD]&series=true [ GET: get list of studies/series/images in this Remote PAC. Formats for date filter - on: '20071001', from/to: '20071001-20080220', before: '-20080220', after: '20071001-' . If series= true, return series records as well as studies.]
/pacs/{id}/subjects/?patientNameFilter=[filter]&patientIDFilter=[filter] [ GET: get list of subjects in this Remote PAC.]
/pacs/{id}/subjects/{subjectid}/studies/?studyDateFilter=[YYYYMMDD] [ GET: get list of studies for this subjectid in this Remote PAC.]
/pacs/{id}/subjects/{subjectid}/studies/{studyid}/series/ [ GET: get list of series for this studyid and subjectid in this Remote PAC.]
/pacs/{id}/entities/{id}?projectID={id} [ GET: Transfer this data (and all its children) from this Remote PAC to EPad's dcm4chee and into this project]
/pacs/{id}/autoqueries/ [ GET: List of configured automatic queries for this Remote PAC]
/pacs/{id}/autoqueries/{subjectid} [ GET: Get configured automatic query for this Remote PAC and this subject]
/pacs/{id}/autoqueries/{subjectid}?projectID=[projectID]&studyDate=[YYYYMMDD]&subjectName=[patientName]&modality=[modality] [ PUT: Configure an automatic query for this Remote PAC and this subject where modality is CT, BI, US, PET etc. projectID is required, the other parameters are optional]
/pacs/{id}/autoqueries/{subjectid}?enable=[true|false] [ PUT: Enable/Disable automatic query for this Remote PAC and this subject]
/pacs/{id}/autoqueries/{subjectid} [ DELETE: Delete configured automatic query for this Remote PAC and this subject]

Administration

/epad/status [ GET: get the status of the server ]
/epad/imagecheck [ GET: force reprocessing of all ePAD images ]

/users/{username}?firstname=[firstname]&lastname=[lastname]&email=[email]&password=[password]&oldpassword=[oldpassword]&enable=[true|false]&addPermission=[perm1]&removePermission=[perm2] [PUT: modify existing user (all parameters are optional, old password is required only when modifying current password). /users/{username | email}/resources/{id}/files [ GET / POST: get the files / create a new file ]
/users/{username | email}/resources/{id}/files/{id | filename} [ GET / DELETE: get / delete a resource file ]
